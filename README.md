# Wisteria KDE

**Wisteria KDE** is a simple and minimal theme inspired by **Orchis Dark** and **Adwaita GTK**.

A project with no grand ambitions - simply a way to preserve my simple theme. It remains under active development, with new improvements and refinements being added over time.

> **Note:**
> Some features are still being polished. Aurorae might be replaced in the future.

## Installation

There isn't an installation script per se yet. But if you prefer to do it manually instead of using the KDE Store, clone the repository and create the symbolic links:

```bash
git clone https://github.com/ryanmferreira/wisteria-kde-theme.git
cd wisteria-kde-theme
./create-symbolic-links.sh
```
